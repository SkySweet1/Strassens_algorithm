## Содержание

Проект содержит:
- спецификации исходной программы и подпрограмм (входные параметры, выходные параметры, их тип, допустимый диапазон значений и т.д.)
- расставленые комментарии по телу программного кода, которые позволяют определять, какие именно действия выполняются на данном шаге;
- подпрограммы возвращают информацию об успешном выполнении либо код ошибки, если она возникает (в частности, приведено описание этих кодов), а вызывающая программа/процедура проверяет возвращаемый код ошибки;
- массивы более 16Кбайт создаются динамически по ходу выполнения программы, а перед завершением программы/подпрограммы выделенная под динамические массивы память освобождается;
- проведенные экспериментальные исследования эффективности работы алгоритма Штрассена, в том числе в зависимости о размера матриц и порядка поля;
- проведены экспериментальные исследования по сравнению эффективности различных вариантов алгоритма Штрассена, а так же выбран оптимальный и обоснован вариант выбора.

## Использование

```bash
    gcc main.c -o data && ./data
```
